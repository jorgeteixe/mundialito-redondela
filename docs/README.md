# Documentation Directory 📖

This directory contains technical guides, workflow documentation, and development instructions for the Mundialito Redondela monorepo.

## What Belongs Here

- **Workflow Guides**: Step-by-step instructions on automated tasks (e.g., Open Graph generation, visual regression tests).
- **Feature Design**: Specifications, architectural patterns, and structural overviews of the platform's layout and data flows.
- **Setup Manuals**: Guides detailing environment configurations, API integrations, and developer toolchain settings.

## Folder Conventions

- All files must be written in English.

- Keep files focused, modular, and up-to-date as the tooling and implementation details evolve.
